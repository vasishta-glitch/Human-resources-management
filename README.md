## 📌 How to Use

1. Click on the subject folder link in the table above.
2. Inside each folder you will find the PDFs and/or PPTs for that subject.
3. Click any file to view it directly in the browser, or download it.

---

## 📥 Adding New Materials

Place new PDF or PPT files in the corresponding subject folder and commit them to this repository. GitHub Copilot can then reference these files by their path (e.g. `02-Recruitment-and-Selection/lecture-1.pdf`).
